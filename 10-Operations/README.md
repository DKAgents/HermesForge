# 10-Operations

This folder contains operational standards, templates, and guidelines for Hermes and any automated processes in the HermesForge system.

## Purpose

- Define consistent patterns for automation
- Make it easy for Hermes to create reliable scripts and cron jobs
- Document how Git, cron jobs, logging, and scripts should be handled

## Key Documents

- [automation-standards.md](./automation-standards.md)
- [cron-job-template.md](./cron-job-template.md)
- [git-workflow-standards.md](./git-workflow-standards.md)

## Usage

Hermes (and future agents) should reference these documents when creating new automation, scripts, or scheduled tasks.
