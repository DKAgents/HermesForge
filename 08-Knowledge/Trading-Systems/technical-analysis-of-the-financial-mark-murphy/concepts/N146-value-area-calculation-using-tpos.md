# Value Area Calculation Using TPOs

The Value Area is determined each trading session by price rotations around the modal price (the price with the highest TPO count or the fairest price). It is computed by counting 70% of all TPOs surrounding the fairest price, representing an estimate of fair value approximated by one standard deviation of the session's trading volume. When a longer term trader makes a trade in the value area, he is buying low or selling high in relation to a longer term view, not in relation to today's value, creating an imbalance in today's value area.


## Direct Quotes

> "The value area is an estimate of fair value which is approximated by one standard deviation of the session's trading volume." — John J. Murphy
