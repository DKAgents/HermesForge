# Next Contract — Definition

A Next Contract is identical to the Nearest Contract except it always uses data from the contract that follows the Nearest Contract. For example, if the Nearest Contract uses December T-Bond data, the Next-1 Contract uses March T-Bond data. A Next-2 Contract uses data from two contracts beyond the Nearest, providing analysts with forward-looking contract data for spread analysis or system testing.
