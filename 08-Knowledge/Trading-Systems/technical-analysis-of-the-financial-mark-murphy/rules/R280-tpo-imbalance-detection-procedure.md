# TPO Imbalance Detection Procedure

To determine which side contains the longer term imbalance: 1) draw a line through the fairest price, and 2) count TPOs on either side of the fairest price until a single print is encountered. The imbalance is assigned to the side with the smaller number of TPOs because longer term trader activity represents the smaller percentage of total trade in the value area. For example, if the TPO count was 22 above and 12 below the fairest price, that would indicate net TPO selling with a mild bias toward lower prices.
