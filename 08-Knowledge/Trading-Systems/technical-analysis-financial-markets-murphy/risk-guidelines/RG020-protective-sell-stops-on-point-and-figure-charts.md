# Protective Sell Stops on Point and Figure Charts

On a more sensitive (smaller box size) point and figure chart, traders can place protective sell stops below the latest columns of o's. The increased number of o columns generated by a smaller box size provides more frequent and precise stop placement opportunities.


## Direct Quotes

> "It also allows the trader to raise protective sell stops below the latest columns of o's." — John J. Murphy
