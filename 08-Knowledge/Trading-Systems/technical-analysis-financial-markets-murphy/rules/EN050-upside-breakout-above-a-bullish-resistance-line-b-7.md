# Upside Breakout Above a Bullish Resistance Line (B-7)

This signal requires two elements: a buy signal must have already been given, and the upper channel line must be completely cleared. Both conditions must be satisfied simultaneously for this pattern to be valid.


## Direct Quotes

> "A buy signal must have already been given; and the upper channel line must be completely cleared." — John J. Murphy
