# Return Move After Double Top Breakdown

After a double top breaks down through the support at point B, a return move back to the breakout point is not unusual before the downtrend resumes. Traders should be aware of this potential retest and not mistake it for a reversal of the reversal.


## Direct Quotes

> "A return move to the breakout point is not unusual prior to resumption of the downtrend." — John J. Murphy
