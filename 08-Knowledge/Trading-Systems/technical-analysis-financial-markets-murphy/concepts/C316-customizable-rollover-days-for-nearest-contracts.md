# Customizable Rollover Days for Nearest Contracts

Traders can customize the number of days before expiration at which they roll over their Nearest Contract, allowing flexibility based on liquidity preferences and trading style. Examples include rolling at expiration (TRNE00), 15 days prior (TRNE15), or at various other intervals (5, 12, 21 days). All such contracts contain actual contract data; only the source contract changes.
