# Book Slug & Folder Naming Convention

## Rule
`<title-slug>-<author-last-name>` where:
- `title-slug` = title lowercased, non-alphanumeric → hyphens, max 40 chars
- `author-last-name` = author's last name, lowercased

## Examples

| Book | Author | Folder Name |
|---|---|---|
| Trading in the Zone | Mark Douglas | `trading-in-the-zone-douglas` |
| How to Make Money in Stocks | William O'Neil | `how-to-make-money-in-stocks-oneil` |
| Trend Following | Michael Covel | `trend-following-covel` |
| The New Market Wizards | Jack Schwager | `the-new-market-wizards-schwager` |
| Options as a Strategic Investment | Lawrence McMillan | `options-as-a-strategic-investment-mcmillan` |

## Notes
- Apostrophes and special characters are removed (O'Neil → oneil)
- "The" prefix is kept (matches title exactly)
- If author has compound last name, use full last name hyphenated: `van-tharp`
